# Dr. Fixit - AI Agent Health Monitor & Repair System

**OpenFang + MLX Edition** | **Apple Silicon Optimized**

Dr. Fixit is an autonomous Python agent that monitors, repairs, and maintains your OpenFang AI agents running on Mac Mini M4 Pro with MLX acceleration.

---

## 🎯 Features

| Feature | Description |
|---------|-------------|
| **🔍 Health Monitoring** | Hourly checks of all OpenFang agents |
| **🔧 Auto-Repair** | Detects issues and automatically fixes them |
| **📝 Obsidian Logging** | All issues logged to markdown files |
| **💾 Git Backup** | Agent states backed up to this repo |
| **🔔 Notifications** | Desktop alerts when issues are found |
| **⚡ MLX Optimized** | All models configured for Apple Silicon |

## ⏰ Automated Hourly Monitoring

Dr. Fixit runs automatically every hour via cron:

```cron
0 * * * * cd ~/agents-dr.fixit && python3 dr-fixit/src/dr_fixit.py
```

### What It Does Every Hour:

1. ✅ **Check Daemon Health** - Is OpenFang running?
2. ✅ **Check All Agents** - Are all 13 agents responsive?
3. ✅ **Activate Idle Hands** - Makes idle agents accessible
4. ✅ **Log Issues** - Creates Obsidian markdown files
5. ✅ **Backup States** - Saves to `openfang-memory/`
6. ✅ **Git Commit** - Pushes changes to repo
7. ✅ **Notify** - Desktop notification if issues found

## 🔧 Manual Repair Commands

### Restart Daemon:
```bash
openfang stop && sleep 2 && openfang start
```

### Restart Specific Agent:
```bash
openfang agent restart [name]
```

### Activate Hand:
```bash
openfang hand activate [hand-name]
```

### Check Status:
```bash
openfang status
openfang hand active
```

## 🐛 Troubleshooting

### Issue: "Daemon not responding"

**Fix**: Dr. Fixit auto-restarts it, or manually:
```bash
openfang stop && openfang start
```

### Issue: "MLX not working"

**Fix**: Re-run optimizer:
```bash
python3 dr-fixit/src/mlx_optimizer.py
```

### Issue: "Agents not accessible"

**Fix**: Check active hands:
```bash
openfang hand active
```
